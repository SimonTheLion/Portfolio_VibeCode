## How to Request Features

### Simple Request:
> "Add [feature name] to my portfolio"

### With Details:
> "Add [feature name] with [specific requirements]. It should [describe behavior]."

### Examples:

**Dark Mode:**
> "Add a dark mode toggle button in the navigation. It should remember the user's preference and have smooth transitions."

**Back to Top:**
> "Add a back-to-top button that appears when scrolling down. It should be in the bottom-right corner and smoothly scroll to top."

**Scroll Progress:**
> "Add a scroll progress indicator bar at the top of the page showing how far the user has scrolled."
